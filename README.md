# Music Dashboard

A premium Flutter Web music analytics dashboard featuring:
- **Deep Dark Theme**: Custom `Color(0xFF0F111A)` styling.
- **Glassmorphism**: Sidebar and UI elements use real-time blur.
- **Firebase Authentication**: Secure Google Sign-In with allow-list restriction.
- **Performance**: Optimized for Web with CanvasKit/Wasm support.

## Access
Restricted to authorized administrators only.
